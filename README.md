# 🎓 无知之幕 · 重回高考

> *Veil of Ignorance: Return to Gaokao*

**随机重生到1977-2026任意一年，随机落在中国任意一个省份。看看你能考上哪所大学。**

[🔗 立即体验](https://lovebicheng-arya.github.io/gaokao-fate/)

## 这是什么？

1977年恢复高考，570万人走进考场，录取27万——**百里挑四**。

2026年，第50届高考。1290万考生。

同样的分数，在不同年代、不同省份，是完全不同的人生。

这个页面让你随机抽取一张「命运卡」——你会被分配到哪个年代？哪个省份？985率1.1%的河南，还是5.8%的北京？录取率4.8%的1977，还是92.9%的2021？

**141张卡**覆盖中国全部31个省份 × 4个关键年代，外加**12张隐藏名人卡**（屠呦呦、袁隆平、雷军、周杰伦、刘慈欣、全红婵……）和极低概率的稀有彩蛋。

## 🎯 玩法

1. 打开页面，深色遮罩浮起，卡片弹出
2. 看你的年代 + 省份 + 录取率 + 大学
3. 截图分享到小红书/朋友圈/X
4. 再抽一次——看能不能抽到名人卡
